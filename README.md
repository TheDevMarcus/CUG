**This project has not been updated since 2024.**

Please be aware that this codebase is essentially **abandonware** at this point. It remains here for historical and reference purposes only.

## Current State of the Project

If you're looking at this code, you should know:

### 🎨 Terrible UI
- The user interface is dated and not user-friendly
- Basic styling that hasn't evolved since 2024
- Layout issues on modern browsers and devices
- Minimal consideration for user experience

### 🐛 Known Bugs
This project is riddled with issues, including but not limited to:
- Various functionality bugs throughout the site
- Unoptimized performance
- Potential broken links and features
- Inconsistent behavior across different browsers

### 🚧 Not Production Ready
This code should **NOT** be used in any production environment. It was a passion project that served its purpose at the time but has since been left untouched.
